Naming Template

URL Pattern is required.
The URL search pattern for matching web pages. The wildcard character * can be used. For example:
  • https://www.example1.com/*
  • https://www.example2.com/photo/*
  • https://www.example3.com/*/photo?id=*
The full path to the directory on FileLu where the target image will be saved. WIC parameters are supported. The path must start with the / character. For example:
  • /{host}
  • /example1/{now-YYYYMM}
  • /example2/img{now-YYMM}_{path-1}
Leave blank to save files to the fallback directory.
The image file name template without extension name. WIC parameters are supported. For example:
  • photo{now}
  • {now-MMDD}{path-2}
  • image_{query-id}
Leave blank to use fallback file name.
