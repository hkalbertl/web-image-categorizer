Naming Template

URL Matching is required.
The search text for matching web pages. Wildcard character * can be used. For example:
  • https://www.example1.com/*
  • https://www.example2.com/photo/*
  • https://www.example3.com/*/photo*
Full path to the directory on your storage provider for saving target image. WIC parameters supported. For example:
  • /{host}
  • /example1/{now-YYYY}
  • /example2/photo_{path-1}
Leave blank to save files at the root level of the storage provider.
The image file name template without extension name. WIC parameters supported. For example:
  • photo{now}
  • {now-MMDD}{path-2}
  • image_{query-id}
Leave blank to use current date/time, i.e. 20241231235958, as file name.
